
# Redis Cache

## O que é Redis?

Redis é um banco de dados em memória de código aberto, estruturado como armazenamento de chave-valor. É amplamente utilizado como cache para melhorar o desempenho de aplicações, armazenando dados frequentemente acessados na memória RAM.

## Características Principais

- **Armazenamento em Memória**: Acesso ultrarrápido aos dados
- **Estruturas de Dados Variadas**: Strings, listas, conjuntos, hashes, sorted sets
- **Persistência Opcional**: Pode salvar dados em disco
- **Replicação e Cluster**: Suporta alta disponibilidade
- **TTL (Time To Live)**: Expiração automática de chaves

## Casos de Uso

- Cache de sessões de usuário
- Armazenamento de dados temporários
- Fila de mensagens
- Ranking e contadores em tempo real
